# Inventory Control

## 概要
外部案件を想定した在庫管理システムです。オフィスなどで発注した在庫の名前、数量、保管場所、日時、備考を管理する事ができます。
```
→CodeIgniterによるPHPのCRUD開発
→ログイン・会員登録機能
→在庫管理機能
→JavaScriptによる検索機能
→adminLTE使用
→Illustratorによるロゴ制作
```

<br>

## 環境構築
```
→ apache 
→ PHP 7.4.4
→ Mysql(MariaDB)
→ Codeigniter 3.2.0
```
